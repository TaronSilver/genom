# genom-2

RUNNING THE PROGRAM
(from your genom-2 folder)


Step 1: Make a build folder and build the program
 
 
--> mkdir build 

--> cd build

--> cmake ../

--> make

documentation		--> make doc


Step 2: Execute the program 


with Qt  			--> ./genom-2

with terminal 		--> ./Main

tests 				--> ./gtests
